JavaScript Statements

A JavaScript program is a list of statements to be executed by a computer.

Um programa de computador é uma lista de instruções a ser executada por um computador.
Em linguagem de programação essas instruções são chamadas de statements.
Os statements de JavaScript são compostos de: Valores, operadores, expressoes, palavras chaves e comentários

ponto e vírgula (semicolons)

Ponto e vírgula(;) separam statements em JavaScript

Adicione um ponto e vírgula no fim de cada statement executável

Quando separadas por ponto e vírgula, multiplos statements podem ser declarados em uma linha EX:
let a, b, c;
a = 5; b = 6; c = a + b;

On the web, you might see examples without semicolons. Ending statements with semicolon is not required, but highly recommended.

JavaScript code Blocks

JavaScript statements podem ser agrupados em blocos de códigos por meio do uso de chaves. Esse recurso é utilizado quando queremos executar um bloco de código em conjunto. Tal prática é geralmente usada quando aplicamos funções aos nossos códigos.

JavaScript Keyword(palavras-chave)

JavaScript statements frequentemente começam com palavras chaves para indicar uma ação a ser realizada
