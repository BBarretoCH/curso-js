My first Paragraph

Changing the innerHTML property of an HTML element is a common way to display data in HTML.
Podemos atribuir a mudança a uma variável, quando feito isso, devemos mudar o conteudo da variável. EX:
let a = document.getElementById('demo') a.innerHTML = 'Paragraph Changed'
Isso significa que o conteúdo HTML com id='demo' foi colocado dentro da variável a e em seguida teve seu conteúdo alterado por a.innerHTML = ''.
