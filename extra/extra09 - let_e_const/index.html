JavaScript let

Variáveis definidas com let não podem ser redeclaradas.
Variáveis definidas com let devem ser declaradas antes do uso.
Variáveis definidas com let tem escopo de bloco

Utilizando a palavra var para declarar uma variável nós podemos redefinir seu valor. Ex:
var x = "Bruno Barreto";
var x = 0;
Essa prática é permitida

Escopo de bloco (let e const)

Uma variável com escopo de bloco é utilizada somente no bloco ao qual ela está contida e foi declarada. Ex:

{ 

   let x = 2;
} x não pode ser usado aqui (fora do bloco das chaves)

Variáveis definidas com var não possuem escopo de bloco e podem ser utilizadas em qualquer lugar do código independente de onde tenham sido declaradas (dentro ou fora de blocos)

É importante ficar atento pois se uma variável com var for declarada fora de um bloco e posteriomente tiver um novo valor atribuído a ela dentro de um bloco, essa variável terá o valor que foi atribuído por último, independente se tal evento ocorreu dentro ou fora de um bloco. Ex: 

      var  x = 10;
    // Aqui x é 10
            
    {
        var x = 2;
        // Aqui x é 2
    }
    
    // Aqui x é 2

Se uma variável ja foi declarada, esta não pode ser redeclarada dentro de um bloco

JavaScript const

Variáveis definidas com const não podem ser redeclaradas.
Variáveis definidas com const não pode ser reatribuídas.
Variáveis definidas com const tem escopo de bloco

Exemplo

        const PI = 3.141592653589793;
        PI = 3.14;      // This will give an error
        PI = PI + 10;   // This will also give an error

Variáveis definidas com const devem ter um valor atribuído quando forem declaradas

Exemplo

        
            Correct
            const PI = 3.14159265359;
        

        
            InCorrect
            const PI;
            PI = 3.14159265359;

When to use JavaScript const?

Sempre declarar uma variável com const quando tivermos certeza que o seu valor não irá se alterar

Usar const quando declarar:

  • Uma nova array
  • Um novo objeto
  • Uma nova função
  • Uma nova expressão regular (RegExp)
